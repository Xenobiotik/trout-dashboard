# Факторная модель клева ручьевой форели

## Общая логика

Ручьевая форель чувствительна к температуре воды, кислороду, прозрачности, уровню воды, освещенности, сезонным циклам и резким изменениям погоды. Для дашборда важно разделить факторы на две группы:

- факторы, которые можно мониторить напрямую из открытых источников;
- факторы, которые придется оценивать косвенно через погоду, сезон, осадки и локальные наблюдения.

Индекс клева должен учитывать не только текущее состояние, но и динамику: резкое изменение давления, длительные дожди, потепление после холода, падение уровня воды после паводка, стабильность температуры.

## Основные факторы

| Фактор | Влияние на клев | Как измерять | Доступность для MVP |
|---|---|---|---|
| Температура воздуха | Косвенно влияет на температуру воды, активность насекомых и обмен веществ рыбы | Погодный API | Высокая |
| Температура воды | Один из ключевых факторов активности форели | Датчики, гидропосты, модель по воздуху | Низкая/средняя |
| Давление | Важны резкие изменения и стабильность | Погодный API | Высокая |
| Тренд давления | Резкие скачки могут снижать активность | Расчет по истории давления | Высокая |
| Осадки | Небольшой дождь может улучшать клев, ливни и паводок ухудшают | Погодный API | Высокая |
| Накопленные осадки | Оценивают риск подъема воды и мутности | Расчет за 24-72 часа | Высокая |
| Уровень воды | Ключевой гидрологический фактор | Гидропосты/API, косвенно по осадкам | Средняя/низкая |
| Расход воды | Показывает силу течения и паводковость | Гидрологические модели/API | Средняя |
| Мутность | Высокая мутность мешает визуальной охоте форели | Датчики, спутниково/косвенно по осадкам | Низкая |
| Ветер | Влияет на заброс, насекомых, охлаждение воды | Погодный API | Высокая |
| Облачность | Уменьшает настороженность рыбы, меняет освещенность | Погодный API | Высокая |
| Солнечная радиация | Влияет на прогрев и освещенность | Погодный API | Средняя/высокая |
| Время суток | Утро/вечер часто перспективнее яркого дня | Расчет локального времени | Высокая |
| Длина дня | Сезонный индикатор активности и кормовой базы | Астрономический расчет/API | Высокая |
| Фаза луны | Возможный слабый/спорный фактор, полезен как вторичный сигнал | Астрономический API/расчет | Высокая |
| Лунная освещенность | Может влиять на ночную активность и осторожность | Астрономический API/расчет | Высокая |
| Сезон | Нерест, прогрев, вылет насекомых, летний перегрев | Календарная модель | Высокая |
| Кислород | Критически важен, особенно летом | Косвенно по температуре воды/течению | Низкая |
| Вылет насекомых | Важен для поверхностной активности | Косвенно по температуре, сезону, влажности, ветру | Средняя |
| Антропогенная нагрузка | Давление рыболовов, шум, доступность мест | Пользовательские наблюдения | Низкая |

## Факторы для первой версии индекса

Для MVP лучше взять факторы, которые доступны без сложной инфраструктуры:

1. температура воздуха;
2. расчетная температура воды;
3. давление и тренд давления;
4. осадки за текущие сутки;
5. накопленные осадки за 48-72 часа;
6. ветер;
7. облачность;
8. сезон;
9. время суток и длина дня;
10. фаза луны и лунная освещенность;
11. стабильность условий за последние 24 часа.

## Косвенные оценки

Некоторые важные факторы можно оценивать приближенно:

- температура воды: по сглаженной температуре воздуха за 3-7 дней, сезону и типу водотока;
- мутность: по интенсивности и накоплению осадков;
- уровень воды: по накопленным осадкам и данным гидрологии, если они доступны;
- кислород: по расчетной температуре воды и силе течения;
- активность насекомых: по температуре, ветру, влажности, облачности и сезону.

## Рекомендации по приоритету

Для MVP:

- использовать погодные и астрономические факторы;
- гидрологию представить как расчетный риск;
- не пытаться сразу строить идеальную биологическую модель;
- обязательно показывать уверенность индекса: высокая, средняя или низкая.

Для версии 2:

- подключить реальные гидрологические данные;
- добавить ручной журнал рыбалок;
- обучать веса индекса на фактических результатах;
- добавить разные профили для малых ручьев, средних рек и участков ниже озер.
