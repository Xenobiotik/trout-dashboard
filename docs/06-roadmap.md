## Формат реального погодного снимка

Файл:

```text
data/live-weather-sample.json
```

Источник:

```text
Open-Meteo Forecast API
```

Параметры:

- `past_days=3`;
- `forecast_days=5`;
- `timezone=Europe/Moscow`;
- почасовые переменные: температура, давление, осадки, облачность, ветер, направление ветра, порывы.

Нормализованные текущие поля:

- `airTemperatureC`;
- `pressureHPa`;
- `pressureMmHg`;
- `precipitationMm`;
- `cloudCoverPercent`;
- `windSpeedMs`;
- `windGustsMs`;
- `windDirectionDegrees`;
- `windDirection`.

Дневные агрегаты:

- средняя, минимальная и максимальная температура;
- среднее давление;
- давление в мм рт. ст.;
- сумма осадков;
- средняя облачность;
- средний ветер;
- максимальные порывы;
- среднее направление ветра;
- изменение давления за 24 часа;
- изменение температуры за 24 часа;
- осадки за 24 и 72 часа.

## Формат условий из реальной погоды

Файл:

```text
data/live-conditions-sample.json
```

Генератор:

```text
src/generateLiveConditions.js
```

Логика:

- реальные погодные поля берутся из `data/live-weather-sample.json`;
- расчетная температура воды оценивается по сезонно-инерционной proxy-модели: базовая температура воды по месяцу, сглаженная температура воздуха, мягкая поправка от текущей дневной температуры, сезонные минимумы и максимумы;
- прозрачность воды оценивается через осадки за 24 и 72 часа;
- уровень воды оценивается через накопленные осадки;
- резкие изменения погоды считаются по изменению давления, температуры, ветра, порывов и сильных осадков;
- лунный фактор пока остается нейтральным placeholder до подключения астрономического адаптера.

## Формат live-результатов

Файл:

```text
data/live-results-sample.json
```

Генератор:

```text
src/generateLiveResults.js
```

Этот файл уже содержит индексы, рейтинги, факторы, драйверы, рекомендации и предупреждения, рассчитанные по реальной погоде.

Для локального интерфейса эти данные встраиваются в:

```text
app/live-data.js
```

Генератор:

```text
src/generateLiveAppData.js
```
