# Дашборд клева ручьевой форели

## Как открыть

Открой файл:

```text
app/index.html
```

## Как пользоваться

Открой дашборд и нажми `Реальная погода`.

Теперь дашборд сам запрашивает Open-Meteo в браузере и пересчитывает индекс. Если интернет недоступен, он использует последние сохраненные live-данные.

## Ручное обновление сохраненных live-данных

Этот способ больше не обязателен для обычного пользования, но полезен как запасной вариант:

1. дважды нажать `Обновить погоду.command`;
2. дождаться сообщения `Готово`;
3. обновить страницу дашборда;
4. выбрать режим `Реальная погода`.

Альтернативно через терминал:

```bash
cd "/Users/xenobiotik/Desktop/Codex/trout-dashboard"
node src/updateLiveData.js
```
